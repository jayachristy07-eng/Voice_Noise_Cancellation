# Voice Noise Cancellation
This project is a Python-based Voice Noise Cancellation System.
## Features
- Real-time microphone noise cancellation
- Audio file noise reduction
- Waveform visualization using matplotlib

## Technologies Used
- Python
- NumPy
- SoundDevice
- NoiseReduce
- SoundFile
- Matplotlib

## How to Run
Install libraries:

```bash
pip install numpy sounddevice noisereduce soundfile matplotlib
python main.py
